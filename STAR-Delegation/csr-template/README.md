## Build instructions

Precondition: a fairly recent (1.11+) Golang build environment

- Fetch dependencies
```
go get github.com/xeipuuv/gojsonschema
```

- Build the executable
```
go build
```

- Run tests
```
go test -v
```

- Run it
```
./csr-template
```
